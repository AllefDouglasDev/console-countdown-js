
# ⏱ Terminal Countdown Timer

A simple Node.js countdown timer that displays big ASCII digits in the terminal.

---

## 💡 How to Use

```bash
node timer.js <hours> <minutes> <seconds>
```

Example:

```bash
node timer.js 0 5 0
```

If no arguments are provided, it defaults to 5 minutes.
